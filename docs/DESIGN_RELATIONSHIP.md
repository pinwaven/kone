# Design & Implementation Sync

The project maintains a close relationship between the `design/` folder and the production code.

## Data Analysis Prototypes
- **Python Scripts:** `design/data/analyzer.py` and others are used to prototype signal processing algorithms.
- **Validation:** Algorithms verified in Python are manually ported to `AppCardUtils.kt` in the Android app.

## Specs & Assets
- **Excel Templates:** Used for defining reagent card parameters and formula coefficients.
- **Assets:** UI design mockups (`.jpg`) are stored in `design/` for reference during UI development with Jetpack Compose.
