# GEMINI.md

This file contains foundational mandates and project-specific instructions for the Gemini CLI agent.

## Project Context
- **Project Name:** Kone (Biomarker Analyzer Kino-One)
- **Type:** Android Application

## Engineering Standards
- NEVER perform automatic git commits or pushes. The user will handle all version control operations manually.
- Adhere to Android development best practices.
- Documentation should be stored in the `docs/` folder in Markdown format.
- Design assets and data analysis scripts are located in the `design/` folder.
